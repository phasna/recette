## 🧪 COMMENT TESTER

### Étape 1 : Rechargez

```
Ctrl + Shift + R  (Windows/Linux)
Cmd + Shift + R   (Mac)
```

### Étape 2 : Testez en Français

```
1. Allez sur Vide-Frigo 🧊
2. Ajoutez "Poulet"
3. Cliquez sur une recette
4. Popup s'ouvre avec :
   - Interface en français
   - Ingrédients en anglais (ex: "Chicken", "Onion")
   - Instructions en anglais (ex: "Heat the pan", "Add chicken")
```

### Étape 3 : Changez en Khmer

```
1. Header → Sélecteur langue 🌐 → KM
2. Interface devient en khmer
3. Cliquez sur une recette à nouveau
```

### Étape 4 : Résultat Attendu

```
Popup s'ouvre avec :
- Interface en khmer ✅
- Ingrédients traduits ✅
- Instructions traduites ✅

Exemple :
🥘 គ្រឿងផ្សំ:
• សាច់មាន់ (chicken traduit)
• ខ្ទឹមបារាំង (onion traduit)
• ប៉េងប៉ោះ (tomato traduit)

👨‍🍳 វិធីធ្វើ:
1. ដាំឱ្យក្តៅ ខ្ទះ (heat pan traduit)
2. បន្ថែម សាច់មាន់ (add chicken traduit)
3. ចំអិន 10 នាទី (cook 10 minutes traduit)
```
